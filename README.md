# 🛒 E-Commerce Website

A simple and responsive E-Commerce website built using **HTML, CSS, and JavaScript**.  
This project demonstrates a basic online store interface where users can browse products and explore a modern UI design.

🔗 **Live Demo:**  
https://shamim-01.github.io/e-Commarce/

---
## ✨ What it is

StyleShop is a polished e-commerce demo site created for practice and GitHub. It showcases a shopping homepage with featured products, deals, a
responsive navigation menu, and a sleek cart sidebar experience.
---
## 🚀 Features

- Product showcase layout
- Clean and simple interface
- Interactive JavaScript functionality
- Clean, responsive design for desktop and mobile
- Interactive product filter buttons
- Slide-out cart sidebar UI
- Search box and mobile menu toggle
- Icon-rich layout using Font Awesome
- Google Fonts typography for a modern look

---

## 💡 Good for

- Frontend practice with HTML/CSS/JS
- Building a brandable store landing page
